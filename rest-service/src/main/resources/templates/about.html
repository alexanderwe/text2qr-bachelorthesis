About Text2QR-Translator All about languages

Text2QR Translator is a free service to create translations of texts. The translations are stored in our webservice an can be accessed with QRCodes which are generated through our client. Simply create an encrypted translation with powerful translation APIs. Then place the QR Code on any document you want. Only person who have the QR Code can access the translation.
