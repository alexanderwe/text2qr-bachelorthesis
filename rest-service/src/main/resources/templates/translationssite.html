Get information about specific translation

GET api/translations/{id}

Paths parameters

Parameter Function
translation_id The unique id for the translation

Response body parameter

Parameter Type Function
id Integer An integer that uniquely identifies this list
fromContent String The source text
fromContent String The translated text
fromLang String Language identifier for the source language
toLang String Language identifier for the target language
match double Identifier for how good the translation is
Example request
GET api/translations/3 HTTP/1.1
Host: localhost:8080
Content-Type: application/json
Cache-Control: no-cache
Example response
HTTP/1.1 200 OK
Content-Type: application/json;charset=UTF-8
Date: Fri, 26 Feb 2016 23:24:11 GMT
Server: Apache-Coyote/1.1
Transfer-Encoding: chunked

{
  "id": 3,
  "fromContent": "Hallo",
  "toContent": "Hello",
  "fromLang": "de",
  "toLang": "en",
  "match": 0.85
}

Create a new translation

POST api/translations

Request body parameter

Parameter Type Function
id Integer An integer that uniquely identifies this list
fromContent String The text which was translated
fromContent String The translated text
fromLang String Language identifier for the source language
toLang String Language identifier for the target language
match double Identifier for how good the translation is
Example request
POST api/translations HTTP/1.1
Host: localhost:8080
Content-Type: application/json
Cache-Control: no-cache

{
    "fromContent": "Das Leben ist toll!",
    "toContent": "The life is great!",
    "fromLang": "en",
    "toLang": "de",
    "match": 0.96
}
