System requirements

Download our client program on your computer to use our service. You simply need to create an account on our homepage and download the client. We do not offer an possibility to create translation and QRCodes in the web interface, due to high security risks when creating encrypted translations.

Operating system Windows, MacOS, Linux
Memory 300mb
Java Version 8
Internet connection Required
Text2QR Account Required

You can choose which version you want to download. The .jar file can be run on every operating system. But be aware that you can not start the .jar on a windows system by double-clicking it. On a MacOS or other Unix systems you can start the program by doble-clicking the .jar file. The .exe is only applicable for Windows systems and the .app Version only for MacOS. But keep in mind the client is not signed, so it could appear that your OS warns you that the program can harm your system.

Be aware that the .app file need to have permission to "control" your mac. Before the first start you have to set this permission in the system preferences --> pricacy --> operational tools --> add text2qr.app!
